Fakes local environment data when launching Minecraft to make it think the location of Appdata, and thus your .minecraft folder, is anywhere you want. Has options and functions to manage profile folders that serve as temporary appdata locations to point Minecraft at, thus giving you simple, quick access to any copy of Minecraft, in any condition or version, you want. 

Compiling this project requires .NET 4, and Visual Studio C#.
